程序说明：
（1）程序摘要：GPIO模拟串口
（2）运行芯片：本工程在KL36芯片运行
（3）软件环境：AHL-GEC-IDE，兼容KDS3.2
（4）硬件环境：嵌入式系统实验套件（AHL-MCU-KL36）
（5）硬件连接：见"..\05_UserBoard\user.h"文件,
                           将GEC_39（PTC_NUM|1）引脚与GEC_38（PTC_NUM|2）用导线连接。
（6）功能简述：
			  通过GPIO模拟串口的接收与发送
			  MCU发送端口和引脚号： SUART_TX   (PTC_NUM | 0)  // GEC_40
			  MCU接收端口和引脚号： SUART_RX   (PTC_NUM | 1)  // GEC_39

			  注意：虽然理论上可以发送任意字节数据，但是内存中只申请了1024字节数据
                  用户可以在includes.h文件中修改大小，最多申请4096个字节（板子RAM 8K）
