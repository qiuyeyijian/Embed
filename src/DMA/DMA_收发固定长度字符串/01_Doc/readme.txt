程序说明：
（1）程序摘要：DMA 接收固定长度字符
（2）运行芯片：本工程在KL36芯片运行
（3）软件环境：AHL-GEC-IDE，兼容KDS3.2
（4）硬件环境：嵌入式系统实验套件（AHL-MCU-KL36）
（5）硬件连接：见"..\05_UserBoard\user.h"文件
（6）功能简述：
			 串口发送10个字节数据，通过DMA放到目的内存地址内，让后通过DMA发送到串口

【UART硬件连接】
             没有用模拟串口，使用的是板子自带的UART_User串口
